# tdb-exam

para hacer una migración de la base de datos

```
cd tdb-backend
npx prisma migrate dev --name init
```

para iniciar el backend
```
cd tdb-backend
npm run start:dev
```

para iniciar el Frontend
```
cd tdb-frontend
npm run dev
```
